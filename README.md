# 🏦 Smart Loan Prediction System 
📌 Project Overview

Smart Loan Prediction System is a Machine Learning based web application developed using **Python, Flask, and Decision Tree Classifier**.

The system analyzes applicant details such as:

- 👤 Age
- 💰 Income
- 🏦 Loan Amount
- 📈 Credit Score

and predicts whether a loan application is likely to be:
✅ Approved or ❌ Rejected

## ⚙️ Installation

### Move Into Project Folder
cd Loan__prediction

### Install Dependencies
pip install -r requirements.txt
